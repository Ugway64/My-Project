> **Краткое описание:**  
> VR-симулятор базовых интерактивных действий — интерактивная песочница для тестирования механик в VR. Проект представляет собой универсальный фреймворк для VR-разработчиков, демонстрирующий реализацию ключевых взаимодействий: физические кнопки, настройки, стрельба из лука с натяжением тетивы, кузнечный молот, 3 вида огнестрельного оружия с разными типами перезарядки и универсальная система хватов.

## Цели и задачи проекта

* **Тестирование VR-механик:** Предоставление разработчикам готовой песочницы для отработки и демонстрации базовых интерактивных действий в виртуальной реальности.
![Общий вид симулятора](Screenshots/screen2.png)
* **Демонстрация взаимодействий:** Реализация и наглядная демонстрация различных типов взаимодействий:
  * Физические кнопки и базовые настройки.
  * Лук с натяжением тетивы и кузнечный молот.
  * 3 вида огнестрельного оружия с разными типами перезарядки.
![Общий вид симулятора](Screenshots/screen4.png)
* **Универсальность:** Создание отличного фреймворка для VR-разработчиков, который можно использовать как основу для собственных проектов.
* **Изучение VR-взаимодействий:** Знакомство с эргономикой VR-управления, физикой объектов и системами хватов в полностью интерактивной среде.
![Общий вид симулятора](Screenshots/screen3.png)

## Структура проекта

Для поддержания порядка в репозитории используется следующая иерархия папок:

```text
Assets          # Все игровые ресурсы
┣ Models        # 3D-модели интерактивных объектов (оружие, лук, молот и т.д.)
┣ Prefabs       # Преднастроенные игровые объекты (кнопки, оружие, контроллеры)
┣ Scenes        # Игровые сцены (главная сцена-песочница, тестовые сцены)
┣ Scripts       # Исходный код логики симулятора, физики и XR-взаимодействия
┗ UI            # Графические элементы интерфейса, меню настроек и подсказок
```

## 🎮 Схема управления (ввод)

Управление стандартно для VR-контроллеров (совместимо с Oculus Touch, Valve Index, HTC Vive):

| Действие                | Кнопка / Жест                          |
|-------------------------|----------------------------------------|
| **Захват объекта**      | Grip (сжатие)                          |
| **Использование**       | Trigger (курок) — выстрел, натяжение, удар |
| **Перезарядка оружия**  | Кнопка B/Y (правая рука) или X/A (левая) |
| **Перемещение**         | Джойстик / тачпад (плавное движение или телепортация) |
| **Поворот**             | Поворот головы или джойстик (поворот на 45°) |
| **Меню / Пауза**        | Кнопка Menu (левая рука)               |
| **Настройки**           | Кнопка Menu (правая рука) + выбор в интерфейсе |

Все управления можно переназначить через `Input System` в Unity.

## 💻 Требования к системе и оборудованию

### Минимальные требования для ПК:
- **ОС**: Windows 10 64-bit
- **Процессор**: Intel Core i5-4590 / AMD Ryzen 5 1500X
- **ОЗУ**: 8 ГБ
- **Видеокарта**: NVIDIA GTX 1060 / AMD Radeon RX 480
- **VR-шлем**: с поддержкой OpenXR (Oculus Rift, HTC Vive, Valve Index и др.)

### Для Oculus Quest (Android):
- **ОС**: Android 10+
- **VR-шлем**: Oculus Quest / Quest 2 / Quest Pro
- **Объём хранилища**: не менее 1 ГБ свободного места
